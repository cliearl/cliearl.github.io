내가 Android Studio에서 사용하는 Plugins

Jul 25, 2021
1 min read

제가 Android Studio에서 사용하는 Plugin을 정리해 보았습니다.

ADB Wi-Fi

  • https://plugins.jetbrains.com/plugin/14969-adb-wi-fi
  • 단말을 Wi-Fi로 연결해서 디버그할수 있게 해줌

CodeGlance

  • https://plugins.jetbrains.com/plugin/7275-codeglance
  • 에디터 오른쪽에 코드 미니맵을 표시

CPU Usage Indicator

  • https://plugins.jetbrains.com/plugin/8580-cpu-usage-indicator
  • 스테이터스바에 CPU 사용량을 표시

CSV

  • https://plugins.jetbrains.com/plugin/10037-csv
  • CSV 파일을 테이블 형식으로 편집할 수 있게 함

GitToolBox

  • https://plugins.jetbrains.com/plugin/7499-gittoolbox
  • Git status와 blame을 보기 좋게 표시하여 잘 관리할 수 있게 함

IDE Features Trainer

  • https://plugins.jetbrains.com/plugin/8554-ide-features-trainer
  • IDE의 기능을 학습하는 튜토리얼 제공

Indent Rainbow

  • https://plugins.jetbrains.com/plugin/13308-indent-rainbow
  • 각 인덴트에 서로다른 색을 붙여줌. Preferences > Editor > General > Appearance > Show whitespaces와 조합해 사용하면 보기 편함

JSON To Kotlin Class

  • https://plugins.jetbrains.com/plugin/9960-json-to-kotlin-class-jsontokotlinclass-
  • JSON 파일을 코틀린 클래스로 편하게 변환해줌

Key Promoter X

  • https://plugins.jetbrains.com/plugin/9792-key-promoter-x
  • 수행한 조작에 대응하는 단축키를 화면에 표시

Mario Progress Bar

  • https://plugins.jetbrains.com/plugin/14708-mario-progress-bar
  • 프로그래스바를 마리오로 바꿔줌

Markdown

  • https://plugins.jetbrains.com/plugin/7793-markdown
  • 마크다운 파일 표시. 평은 그리 좋지 않음

Presentation Assistant

  • https://plugins.jetbrains.com/plugin/7345-presentation-assistant
  • 수행한 키 조작을 화면에 표시해 줌

Save Actions

  • https://plugins.jetbrains.com/plugin/7642-save-actions
  • 저장할 때 자동으로 실행할 액션을 지정 가능

Statistic

  • https://plugins.jetbrains.com/plugin/4509-statistic
  • 코드 통계를 표시

String Manipulation

  • https://plugins.jetbrains.com/plugin/2162-string-manipulation
  • 문자열에 대해 Case switching, sorting, filtering, incrementing, aligning to columns, grepping, escaping, encoding 등등을 수행

Xcode-Dark Theme

  • https://plugins.jetbrains.com/plugin/13106-xcode-dark-theme
  • 안드로이드 스튜디오를 Xcode 스타일로 점령하는 테마
