VSCODE에서 Python VirtualEnv 환경설정하기

Dec 13, 2020
1 minute read

윈도우 환경에서 VS Code에 파이썬 가상환경 설정하고 사용하는 법에 대해 정리했다.

venv 환경 작성

  • 작업폴더를 연 뒤 터미널에서 다음 값 입력
$ python3 -m venv .venv

  • 상태표시줄의 파이썬을 눌러 ('.venv')인 python을 선택

  • 다음 명령어로 가상환경을 활성화 후 패키지 설치

//윈도우
.venv\\scripts\\activate

// macOS
source .venv/bin/activate

code-runner를 실행하기 위한 settings.json 설정

// 윈도우
"python.pythonPath": "blahblah\\.venv\\Script\\python.exe",

// macOS
"python.pythonPath": ".venv/bin/python",
python virtualenv
